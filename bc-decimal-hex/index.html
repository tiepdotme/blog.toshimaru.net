bcコマンドで16進数変換

05 Dec 2013 linux

パッと計算したいときにはよくコマンドラインを立ち上げてbcコマンドを使います。基本10進数を扱うのがメインなんですが、たまーに16進数とかも扱いたいときがあります。その方法をご紹介。

基本

基本は下記の２つの値を設定して10進数以外を扱うことができます。（初期値は10で設定されてます）

  • obase: 出力したい進数を設定
  • ibase: 入力となる進数を設定

10進数→16進数

obaseを設定すればOK.

obase=16
255
FF

255 -> FF です。

16進数→10進数

逆もいってみます。ibaseを設定します。

ibase=16
AA
170

AA -> 170 です。

2進数、8進数

もちろん他にも2進数やら8進数やら設定次第でOKです。

obase=2
255
11111111

255 -> 11111111 です。
