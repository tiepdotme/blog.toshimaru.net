Rubyでつかわれる%（パーセント）を使った記法、たまに忘れることがあるのでメモがてらにまとめてみる。

| %記法 | %使わない場合の書き方 |
| ---| --- |
| `%`, `%Q` | `""` (ダブルクオート) |
| `%q` | `''` (シングルクオート) |
| `%w` | `[]` (文字列配列)|
| `%W` | `[]` (文字列配列・式展開あり) |
| `%i` | `[]` (シンボル配列) |
| `%I` | `[]` (シンボル配列・式展開あり) |
| `%x` | <code>``</code> (コマンド実行) |
| `%s` | `:` (シンボル) |

※ %の後には丸括弧（`()`）や波括弧（`{}`）が使えます
