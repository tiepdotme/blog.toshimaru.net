Rubyでつかわれる%（パーセント）を使った記法、たまに忘れることがあるのでメモがてらにまとめてみる。

| %記法 | %使わない場合の書き方 |
| ---- | ---------- |
| `%`, `%Q` | `""` (ダブルクオート) |
| `%q` | `''` (シングルクオート) |
| `%w` | `[]` (文字列配列)|
| `%W` | `[]` (文字列配列・式展開あり) |
| `%i` | `[]` (シンボル配列) |
| `%I` | `[]` (シンボル配列・式展開あり) |
| `%x` | <code>``</code> (コマンド実行) |
| `%s` | `:` (シンボル) |
| `%r` | `//` (正規表現) |

※ %の後には丸括弧（`()`）や波括弧（`{}`）などの記号が使えます

文字列表現系
---

### %, %Q

    > %(a b c)
     => "a b c"

    > a, b, c = 1, 2, 3
    > %Q(#{a} #{b} #{c})
     => "1 2 3" # 式展開される

### %q

    > %q(a b c)
     => "a b c"

    > a, b, c = 1, 2, 3
    > %q(#{a} #{b} #{c})
     => "\#{a} \#{b} \#{c}" # 式展開されない

配列表現系
---

### %w

    > %w(a b c)
     => ["a", "b", "c"]

### %W

    > a, b, c = 1, 2, 3
    > %W(#{a} #{b} #{c})
     => ["1", "2", "3"]

### %i

    > %i(a b c)
     => [:a, :b, :c]

### %I

    > a, b, c = 1, 2, 3
    > %I(#{a} #{b} #{c})
     => [:"1", :"2", :"3"]

コマンド実行
---

### %x

    > %x(date)
     => "Sun Jan 11 01:01:52 JST 2015\n"
    > `date`
     => "Sun Jan 11 01:01:54 JST 2015\n"

シンボル
----

### %s

    > %s(a b c)
     => :"a b c"

正規表現
----

### %r

    > %r(^http://)
    => /^http:\/\//
