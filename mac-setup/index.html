Mac購入時に僕がインストールしたアプリや設定を公開するよ

16 Aug 2013 mac

MacのSSDが逝ってしまいデータが全て吹っ飛びました。その際にMacを再セットアップしたので、その時インストールしたアプリや設定を公開してみたいと思います。

ブラウザ

  • Chrome
  • Firefox
  • Opera

Chromeがメインブラウザなのでインストール。Firefoxもたまに使うことがあるので一応インストールしておく。

Operaはオプションでどうぞ。Operaは15でレンダリングエンジンがWebkitベースになるので要らない気もしますが、元Operaユーザとしてインストール。

コミュニケーション

  • Twitterクライアント
  • Airmail
  • Skype

Skypeの代わりにGoogle Hangoutsとか使ってもいいかも。

エディタ

  • Sublime Text 2
  • MacVim

メインエディタであるSublimeをインストール。

クラウドストレージ

  • Evernote
  • Dropbox
  • Google Drive

この辺がメジャーですかね。

開発/Development

  • Xcode
  • Github
  • Source Tree

Githubは基本的にコマンド通して使ってますがGUIバージョンもインストール。Source Treeもgit用に。

フォント

  • Inconsolata
  • Ricty
  • DEJA VU SANS MONO

上２つは有名ですが、最近のお気にはDEJA VU SANS MONO。

ターミナル設定

  • homebrew
  • zsh + oh-my-zsh
  • RVM

homebrewは基本ですね。zshもbrewでインストール。RVMも入れときましょう。

その他

  • Pocket
  • Kindle
  • Google日本語入力
  • Microsoft Office

この辺は各個人使っているサービスに応じて。MS Officeはサブスクリプション形式のoffice365がいいでしょう。

導入見送ったやつ

  • boxen を使った環境構築
  • Tweet
