[scala] play framework with Mac

10 Nov 2012 scala

Macを使い、今一部で流行っているらしいScalaによるWebフレームワーク、playframeworkを動かしてみます。

インストール

homebrewを使えばscalaおよびplayフレームワークは簡単にイントールできます。

brew install play
brew install scala

確認してみます。can play?

play
       _            _ 
 _ __ | | __ _ _  _| |
| '_ \| |/ _' | || |_|
|  __/|_|\____|\__ (_)
|_|            |__/ 

play! 2.0.2, http://www.playframework.org

This is not a play application!

Use `play new` to create a new Play application in the current directory, 
or go to an existing application and launch the development console using `play`.

You can also browse the complete documenta

can scala?

scala -version
Scala code runner version 2.9.2 -- Copyright 2002-2011, LAMP/EPFL

アプリケーション作成

create new play application.

play new first_app

What is the application name? 
> first_app

Which template do you want to use for this new application? 

  1 - Create a simple Scala application
  2 - Create a simple Java application
  3 - Create an empty project

> 1

OK, application first_app is created.

Have fun!

これで起動する準備は完了。

アプリケーション起動

run application.

cd first_app/
play
[first_app] $ run
[first_app] $ run

--- (Running the application from SBT, auto-reloading is enabled) ---

[info] play - Listening for HTTP on port 9000...

http://localhost:9000/ で下記の画面が確認できればOK.

What's next step?
