# blog.toshimaru.net

My blog repository powered by Jekyll.

# Rake Tasks

## Create new post

    $ rake post

or

    $ rake new

Then, title your new post.

## Initial setup

    $ rake setup

## Serve static pages

    $ jekyll serve

## Deploy to gh-pages

    $ rake deploy
