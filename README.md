blog.toshimaru.net
===========
My blog repository powered by Jekyll. 

[![Build Status](https://travis-ci.org/toshimaru/blog.toshimaru.net.png?branch=master)](https://travis-ci.org/toshimaru/blog.toshimaru.net)

Create new post
----
    $ rake post

And then, title new post.

Deploy to gh-pages
----
    $ rake deploy
